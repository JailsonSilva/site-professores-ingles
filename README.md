Sobre: Este é um projeto que visa desenvolver um Site para professores e alunos da língua inglesa, onde eles possam interagir, receber atividades dos professores e compartilhar dúvidas, textos e poemas autorais.O projeto está sendo desenvolvido com o framework Ruby on Rails, com o apoio das ferramentas RubyMine, Cucumber, Capybara e Slack
